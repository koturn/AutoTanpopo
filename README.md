AutoTanpopo
===========

Automated mouse manipulation application to [paste "Tanpopo" (dandelion) on "Sashimi" (sliced raw fish) in VRChat](https://booth.pm/ja/items/3183508 "【UdonChips対応】刺身にたんぽぽを乗せるギミック - Tomentosa Repository - BOOTH").
